# 雷达装备使用手册

## 一、雷达基本参数

### 1.1 核心指标
- **探测距离**：取决于发射功率、天线增益、目标RCS、信噪比门限
- **雷达方程**：R_max = (P_t * G^2 * λ^2 * σ / ((4π)^3 * S_min))^(1/4)
  - P_t: 发射功率(W)
  - G: 天线增益
  - λ: 波长(m)
  - σ: 目标RCS(m²)
  - S_min: 最小可检测信号功率(W)

### 1.2 扫描参数
- **方位覆盖**：天线水平扫描范围（典型±60°）
- **俯仰覆盖**：天线垂直扫描范围（典型±30°）
- **扫描周期**：完成一次全范围扫描的时间
- **指向模式**：相对指向（随机体转动）或绝对指向（固定方向）

### 1.3 信号特征
- **工作频率**：标称频率(MHz)
- **信号类型**：普通脉冲/线性调频/非线性调频/相位编码
- **频率类型**：恒频/频率分集/频率捷变
- **PRF类型**：恒定PRF/PRF抖动/PRF滑移
- **极化方式**：圆极化/椭圆极化/线极化/垂直极化/水平极化

## 二、操作规范

### 2.1 开机流程
1. 确认飞行状态稳定
2. 开启雷达电源（TURN_ON）
3. 等待预热（TurnOnTime参数定义的启动时间）
4. 进入搜索模式

### 2.2 搜索策略
- **广域搜索**：大方位角+大仰角，适合态势感知
- **区域搜索**：调整指向（PointingAzimuth/PointingElevation）对特定区域搜索
- **高频搜索**：提高DetectFrequency以加快刷新率

### 2.3 目标探测
- 探测范围受目标RCS影响（RCS越大探测距离越远）
- 大气效应会衰减雷达信号
- 友敌识别：IsDetectFriend参数控制是否探测友方

### 2.4 抗干扰使用
- 遭受干扰时观察信噪比变化
- 可通过频率捷变降低干扰效果
- 必要时关闭雷达改为被动探测

### 2.5 关机时机
- 隐蔽接近敌方时（降低被RWR探测概率）
- 弹药耗尽需撤退时
- 处于己方防空覆盖区内不需要自身探测时
