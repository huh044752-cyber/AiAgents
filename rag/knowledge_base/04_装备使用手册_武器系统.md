# 武器系统使用手册

## 一、空空导弹武器系统

### 1.1 系统组成
- **火控雷达**：负责目标探测和跟踪
- **武器挂架**：挂载导弹
- **火控计算机**：计算发射参数和攻击包线
- **导弹**：主动雷达制导空空导弹

### 1.2 交战模式
| 模式 | 说明 |
|------|------|
| NONE | 空闲状态 |
| SELECT | 目标已选择 |
| ACQUIRE | 目标获取/锁定中 |
| READY_TO_FIRE | 准备发射 |
| FIRE | 发射中 |
| FLYOUT | 导弹飞出 |
| PERSISTFIRE | 持续射击 |

### 1.3 发射流程
1. **目标选择(SELECT)**：从雷达探测目标中选择攻击目标
2. **目标锁定(LOCK)**：雷达转为跟踪模式，锁定目标
3. **包线确认**：确认目标在导弹发射包线内
4. **发射(LAUNCH)**：发射导弹
5. **制导**：部分导弹需要雷达持续照射目标
6. **评估**：确认命中或脱靶

### 1.4 发射包线
- **最大射程**：取决于导弹速度、目标距离和高度
- **最小射程**：导弹需要加速距离
- **目标速度**：追尾发射射程大于迎头发射
- **高度差**：高打低射程增大，低打高射程减小
- **使用CAN_ATTACK查询确认目标是否在包线内**

## 二、操作规范

### 2.1 弹药管理
- 起飞前确认弹药装载量
- 使用HAVE_MUNITION查询剩余弹药
- 保留至少一枚导弹用于自卫
- 多目标交战时合理分配弹药

### 2.2 多目标交战
- 最大同时交战目标数由TargetNum参数决定
- 发射间隔由IntervalTime参数决定
- 使用LAUNCHNUM设置连续发射数量
- 优先攻击高威胁目标

### 2.3 中止交战
- 使用ABORT中止当前交战
- 中止后导弹进入自毁或自主制导
- 可重新选择新目标

## 三、安全约束
- 不得向友方目标发射
- 发射前必须完成敌我识别
- 导弹发射最低高度限制
- 编队中不得穿越友方射界
